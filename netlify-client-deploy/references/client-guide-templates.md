# クライアントへの案内テンプレート

クライアントに送るメッセージの例文集。コピペしてそのまま使える。

---

## 初回案内（アカウント作成 + トークン発行）

```
ホームページの公開にあたり、Netlify（ネットリファイ）というサービスのアカウントを
ご用意いただく必要があります。無料で使えるサービスです。

【手順】
1. 以下のURLにアクセスしてください
   https://app.netlify.com/signup

2. 「Sign up with Google」をクリックして、Googleアカウントでログインしてください

3. ログインしたら、以下の手順でトークン（パスワードのようなもの）を発行してください:
   - 左下のアイコン → 「User settings」
   - 左メニューの「Applications」
   - 「Personal access tokens」の「New access token」
   - 名前は何でもOK（例: deploy-token）
   - 「Generate token」をクリック
   - 表示された文字列（nfp_で始まるもの）をコピーして私にお送りください

⚠️ トークンは1回しか表示されませんので、必ずコピーしてお送りください。

このトークンがあれば、私の方でホームページの公開・更新作業を行いますので、
それ以降はお客様の方で作業していただくことはありません。
```

---

## 更新時の案内（作業不要の連絡）

```
ホームページの更新が完了しました。
以下のURLでご確認いただけます:

https://xxx.netlify.app

何か問題がありましたらお知らせください。
```

---

## 独自ドメイン設定の案内

```
独自ドメインの設定を行います。
お使いのドメイン管理サービス（お名前.com等）で、以下のDNS設定を追加してください:

種類: CNAME
ホスト名: www（またはお好みのサブドメイン）
値: xxx.netlify.app

設定後、反映まで数時間かかる場合があります。
反映されましたら、SSL証明書（https）も自動的に設定されます。
```
